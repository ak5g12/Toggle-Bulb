Toggle Bulb Project

A simple web project built using HTML, CSS, and JavaScript that simulates a light bulb. Clicking the switch turns the bulb ON, and clicking it again turns the bulb OFF.

Features
Turn bulb ON with a click
Turn bulb OFF with another click
Interactive user interface
Beginner-friendly JavaScript project
Technologies Used
HTML5
CSS3
JavaScript
How to Run
Clone or download this repository.
Open the project folder.
Run the index.html file in your web browser.
Project Purpose

This project demonstrates DOM manipulation and event handling in JavaScript by changing the bulb's state when the user interacts with the switch.
